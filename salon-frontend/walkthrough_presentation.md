## 🔐 3. The Backend Engine

Underneath the hood lies a high-performance **Java Spring Boot API**, orchestrating all data seamlessly.

- **Dynamic Data Sync:** Every customer registering through the portal is immediately tied exclusively to your Salon (`getCurrentSalon()`).
- **Complete Admin Panel:** The Super Admin layout instantly syncs and displays registered users via a freshly minted `/api/admin/customers` endpoint. Everything updates in synchronized harmony.
